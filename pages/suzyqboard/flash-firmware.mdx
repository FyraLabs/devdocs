---
title: Flashing Firmware
description: Flash Firmware using the SuzyQ Board
---

When Flashing Firmware, the Fyra SuzyQ Board is no different then using a SuzyQable, so you can just follow [MrChromebox's WP disabling page](https://docs.mrchromebox.tech/docs/firmware/manual-flashing.html).

Note that this WILL work in Ultramarine Linux/Fedora, and you may `dnf install` `flashrom` from Fedora repositories, and `cbfstool` from [Terra](https://terra.fyralabs.com/).

Other known Distros/package managers that package `cbfstool` (as of 1-5-2026):

- Alpine
- AOSC OS
- Arch Linux AUR
- Debian
- NixOS/nixpkgs
- OpenSUSE
- Void Linux
