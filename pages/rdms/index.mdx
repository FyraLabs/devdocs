# Installer and Out Of Box

This section contains documentation for developers or distributions using Readymade and/or Taidan, the installer and out-of-box experience we use in Ultramarine.

If you're trying to install Ultramarine you should check the [Ultramarine wiki](https://wiki.ultramarine-linux.org).

## Glossary

- [Readymade]: The installer app.
- [Taidan]: The out-of-box app.
- [Anaconda]: Fedora's installer (the rhinstaller stack).
- [initial-setup]: Fedora's out-of-box app (rhinstaller).
- [umstellar]: How we installed select drivers before taidan.
- [systemd-repart]: `systemd-repart` creates partition tables, and adds or grows partitions (see [repart.d(5)]).
  It is the main backend used by Readymade.
