# Installation

Katsu is available on the Terra repositories, so you can install it with `dnf`:

```sh
dnf install katsu
```

To enable the Terra repositories, follow the instructions on [Terra's page](../terra/installing)

Katsu is available on [crates.io](https://crates.io/crates/katsu), so you can install it with `cargo`:

```sh
cargo install katsu
```
